Privacy Policy

Last updated: April 30, 2026

Overview

HermesThreads is a personal Threads content research and organization tool. It uses the Threads API to help the authenticated user search public Threads posts by keyword, retrieve post text and metadata, and organize selected results into private research notes.

Information we access

  • Threads account information made available by the Threads API for the authenticated user, such as user ID and username.
  • Public Threads post text and metadata returned by keyword search, such as post ID, username, timestamp, media type, and permalink.
  • Content explicitly created or published by the authenticated user when they request publishing actions.

How information is used

Information is used only to provide content search, research, summarization, and private note organization for the authenticated user.

Data sharing

HermesThreads does not sell user data and does not share collected data with data brokers or advertising networks. Data is not used for public surveillance or profiling products.

Storage

Selected research outputs may be saved in the authenticated user's private note system or local files when requested by the user. The app owner can delete these records on request.

Publishing

The app does not publish posts, replies, or responses unless the authenticated user explicitly requests that specific action.
