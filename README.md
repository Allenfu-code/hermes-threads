# HermesThreads public site

Minimal public website and app assets for Meta Developer App Review.

## Files
- `index.html` — public landing page
- `privacy.html` — privacy policy URL for Meta
- `data-deletion.html` — user data deletion instructions
- `assets/logo-1024.png` — 1024×1024 app icon for Meta
- `assets/logo.svg` — vector logo for the site

## Suggested Meta fields
- App category: Productivity or Business / Utilities, depending on available dropdown options
- Platform: Website
- Privacy Policy URL: `https://<your-domain>/privacy.html`
- Data Deletion URL: `https://<your-domain>/data-deletion.html`
